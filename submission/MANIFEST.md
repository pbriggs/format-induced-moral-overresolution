# Manifest

- manuscript.md: latest uploaded main manuscript markdown.
- supplementary_information.md: supplementary information markdown available in the sandbox.
- latex/main.tex: generated LaTeX source for main manuscript PDF.
- latex/supplementary_information.tex: generated LaTeX source for SI PDF.
- latex/main.pdf: compiled main manuscript PDF, if build succeeds.
- latex/supplementary_information.pdf: compiled SI PDF, if build succeeds.
- figures/: placeholder directories for final/editable/source figure assets.
- source_data/: placeholder directory for release-safe source data.
